!important

A ! El delimitador seguido de la palabra clave importante marca la declaración como importante. La bandera !important altera las reglas seleccionando declaraciones dentro de la cascada. Una declaración que no es importante se llama normal.
Orden
X

Orden

La propiedad CSS order establece el orden para diseñar un elemento en un contenedor flexible o de cuadrícula. Los elementos de un contenedor se ordenan por valor de orden ascendente y, a continuación, por su orden de código fuente.
Herencia
X

Herencia

Antes de empezar, debemos saber que algunas propiedades CSS, como color o font-family, se heredan desde los elementos HTML padres a los elementos HTML hijos, modificando el valor que tienen por defecto.
Colapso de margenes
X

Colpaso de margenes

En CSS, los márgenes adyacentes de dos o más bloques (que podrían o no ser hermanos) pueden combinarse para formar un único margen. Cuando los márgenes se combinan de esta manera decimos que colapsan, y el margen combinado resultante se denomina margen colapsado.
