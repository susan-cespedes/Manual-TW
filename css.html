COLORES

Elemento Descripción Documentación
{;;color;;} Establece el valor de color de primer plano del texto y las decoraciones de texto, establece el valor de color actual color
{;;usando nombres de colores;;} Podemos especificar un color usando un nombre predefinido nombres de colores
{;;funcion RGB (r,g,b);;} Define un color como la mezcla las intensidades de colores primarios, la función: rgb(red, green, blue), valores posibles son: [0-255] o [0%-100%] rgb
{;;valores hexadecimales;;} Número hexadecimal para definir un color. Alternativa a la función rgb, permite definir un color como la mezcla de intensidades de los colores primarios hexadecimal
{;;opacidad;;} Define la transparencia de un elemento,en qué grado se superpone el fondo al elemento opacidad
{;;función hsl(h, s, l);;} Define el color usando la función hsl(hue, saturation, lightness) hsl
{;;hue;;} Valores posibles son: [0-360] Podemos obtener los valores de la rueda de colores hue
{;;saturación;;} Determina que tan fuerte o vivo es un color, valores posibles: [0%-100%] Donde 100% significa completamente saturado (color muy vivo) Donde 0% significa sin saturación alguna (color apagado) saturación
{;;lightness;;} Establece la claridad u oscuridad del color, los valores posibles son: [0%-100%] Donde 100% significa completamente blanco. Donde 50% es normal y Donde 0% es completamente negro lightness
{;;función hsla(h, s, l, a);;} Define el color usando la función hsla(hue, saturation, lightness, opacity), con un valor de opacidad hsla

SELECTORES

Elemento Descripción Documentación
{;;selectores;;} Selecciona elementos del documento a los que queremos aplicar estilos selectores
{;;selector universal (*);;} Selecciona todos los elementos: *{ declaraciones CSS } selector universal
{;;selector de tipo (elem);;} Selecciona elementos usando el nombre del elemento: elem{ declaraciones CSS } selector de tipo
{;;selector de clase (.elem);;} Selecciona elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. .elem { declaraciones CSS } selector de clase
{;;selector id (#elem);;} Selecciona un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, el id debe coincidir exactamente con el valor dado en el selector #valor_atributo { declaraciones CSS } selector id
{;;selector atributos;;} Coincide con los elementos en función de la presencia o el valor de un atributo dado selector atributos
{;;pseudo-classes;;} Selecciona elementos en base a su estado, posicion, e interacción del usuario pseudo-classes
{;;:disabled;;} Representa cualquier elemento deshabilitado :disabled
{;;:first-child;;} Representa el primer elemento entre un grupo de elementos hermanos :first-child
{;;::nth-child;;} Coincide con los elementos en función de su posición entre un grupo de hermanos :nth-child
{;;:hover;;} Se activa cuando el usuario pasa el cursor sobre un elemento (puntero del mouse) hover
{;;pseudo-elementos;;} Selecciona parte de un elemento en lugar del elemento mismo pseudo-elementos
{;;::first-letter;;} Aplica estilos a la primera letra de la primera línea de un elemento de nivel de bloque, pero solo cuando no está precedido por otro contenido ::first-letter
{;;::first-line;;} Aplica estilos a la primera línea de un elemento de nivel de bloque ::first-line
{;;::selection;;} Aplica estilos a la parte de un documento que ha sido resaltada por el usuario (como hacer clic y arrastrar el mouse sobre el texto) ::selection
{;;especificidad;;} Algoritmo utilizado por los navegadores para la declaración CSS más relevante para un elemento especificidad

UNIDADES

Elemento Descripción Documentación
{;;unidades;;} Tipos de valor más utilizados y sus valores y unidades más comunes unidades
{;;integer;;} Es un tipo especial de numero que representa un número entero, ya sea positivo o negativo. Los números enteros se pueden usar en numerosas propiedades de CSS integer
{;;number;;} Representa un número, ya sea un número entero o un número con un componente fraccionario number
{;;dimension;;} Representa un numero con una unidad adjunta, por ejemplo, 10px dimension
{;;percentage;;} Representa un valor porcentual. A menudo se usa para definir un tamaño en relación con el objeto principal de un elemento percentage

PROPIEDADES

Elemento Descripción Documentación
{;;width;;} La propiedad CSS width establece el ancho de un elemento width
{;;min-width;;} La propiedad CSS min-width establece el ancho mínimo de un elemento min-width
{;;max-width;;} La propiedad CSS max-width establece el ancho máximo de un elemento max-width
{;;height;;} La propiedad CSS height especifica la altura de un elemento height
{;;min-height;;} La propiedad CSS min-height establece la altura mínima de un elemento min-height
{;;max-height;;} La propiedad CSS max-height establece la altura máxima de un elemento max-height
{;;padding-top;;} Establece la altura del área de relleno en la parte superior de un elemento padding-top
{;;padding-right;;} Establece el ancho del área de relleno a la derecha de un elemento padding-right
{;;padding-bottom;;} Establece la altura del área de relleno en la parte inferior de un elemento padding-bottom
{;;padding-left;;} Establece el ancho del área de relleno a la izquierda de un elemento padding-left
{;;padding;;} Establece el área de relleno en los cuatro lados de un elemento a la vez padding
{;;border-width;;} Establece el ancho del borde de un elemento border-width
{;;border-style;;} Establece el estilo de línea para los cuatro lados del borde de un elemento border-style
{;;border-color;;} Establece el color del borde de un elemento border-color
{;;border;;} Establece el borde de un elemento (border-width, border-style y border-color) border
{;;margin-top;;} Establece el área de margen en la parte superior de un elemento margin-top
{;;margin-right;;} Establece el área de margen en el lado derecho de un elemento margin-right
{;;margin-bottom;;} Establece el área de margen en la parte inferior de un elemento margin-bottom
{;;margin-left;;} Establece el área de margen en el lado izquierdo de un elemento margin-left
{;;margin;;} Establece el área de margen en los cuatro lados de un elemento margin
{;;box-sizing;;} Establece cómo se calculan el ancho y el alto totales de un elemento box-sizing
{;;aspect-ratio;;} Usada para verificar la relación ancho/alto del viewport aspect-ratio
{;;orientation;;} Usada para verificar la orientación de la pantalla( portrait / landscape) orientation
{;;resolution;;} Usada para verificar la densidad de pixeles del dispositivo resolution
{;;grid;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap grid
{;;@media;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap @media
{;;background-color;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap background-color

FLEXBOX

Elemento Descripción Documentación
{;;display;;} Establece si un elemento de bloque o en línea y si el diseño es de flujo, la cuadrícula o la flexión display
{;;flex-direction;;} Permite establecer la dirección del eje principal del contenedor flexible flex-direction
{;;flex-wrap;;} Controla si el contenedor flexible es de una o de varias líneas y la dirección del eje transversal flex-wrap
{;;flex;;} Establece cómo un elemento flexible crecerá o se reducirá para ajustarse al espacio disponible flex
{;;align-content;;} Ajusta las líneas dentro de un contenedor flexible cuando hay espacio extra align-content
{;;justify-content;;} Define cómo el navegador distribuye el espacio entre y alrededor de los artículos flexibles justify-content
{;;align-items;;} Controla la alineación de los elementos en el eje transversal. En Diseño de cuadrícula, controla la alineación de elementos en el Eje de bloque dentro de su área de cuadrícula align-items
