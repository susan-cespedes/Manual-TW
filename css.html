PROPIEDADES

Elemento Descripción Documentación
{;;width;;} La propiedad CSS width establece el ancho de un elemento width
{;;min-width;;} La propiedad CSS min-width establece el ancho mínimo de un elemento min-width
{;;max-width;;} La propiedad CSS max-width establece el ancho máximo de un elemento max-width
{;;height;;} La propiedad CSS height especifica la altura de un elemento height
{;;min-height;;} La propiedad CSS min-height establece la altura mínima de un elemento min-height
{;;max-height;;} La propiedad CSS max-height establece la altura máxima de un elemento max-height
{;;padding-top;;} Establece la altura del área de relleno en la parte superior de un elemento padding-top
{;;padding-right;;} Establece el ancho del área de relleno a la derecha de un elemento padding-right
{;;padding-bottom;;} Establece la altura del área de relleno en la parte inferior de un elemento padding-bottom
{;;padding-left;;} Establece el ancho del área de relleno a la izquierda de un elemento padding-left
{;;padding;;} Establece el área de relleno en los cuatro lados de un elemento a la vez padding
{;;border-width;;} Establece el ancho del borde de un elemento border-width
{;;border-style;;} Establece el estilo de línea para los cuatro lados del borde de un elemento border-style
{;;border-color;;} Establece el color del borde de un elemento border-color
{;;border;;} Establece el borde de un elemento (border-width, border-style y border-color) border
{;;margin-top;;} Establece el área de margen en la parte superior de un elemento margin-top
{;;margin-right;;} Establece el área de margen en el lado derecho de un elemento margin-right
{;;margin-bottom;;} Establece el área de margen en la parte inferior de un elemento margin-bottom
{;;margin-left;;} Establece el área de margen en el lado izquierdo de un elemento margin-left
{;;margin;;} Establece el área de margen en los cuatro lados de un elemento margin
{;;box-sizing;;} Establece cómo se calculan el ancho y el alto totales de un elemento box-sizing
{;;aspect-ratio;;} Usada para verificar la relación ancho/alto del viewport aspect-ratio
{;;orientation;;} Usada para verificar la orientación de la pantalla( portrait / landscape) orientation
{;;resolution;;} Usada para verificar la densidad de pixeles del dispositivo resolution
{;;grid;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap grid
{;;@media;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap @media
{;;background-color;;} Usada para verificar si el dispositivo usa una pantalla basada en grid o bitmap background-color

FLEXBOX

Elemento Descripción Documentación
{;;display;;} Establece si un elemento de bloque o en línea y si el diseño es de flujo, la cuadrícula o la flexión display
{;;flex-direction;;} Permite establecer la dirección del eje principal del contenedor flexible flex-direction
{;;flex-wrap;;} Controla si el contenedor flexible es de una o de varias líneas y la dirección del eje transversal flex-wrap
{;;flex;;} Establece cómo un elemento flexible crecerá o se reducirá para ajustarse al espacio disponible flex
{;;align-content;;} Ajusta las líneas dentro de un contenedor flexible cuando hay espacio extra align-content
{;;justify-content;;} Define cómo el navegador distribuye el espacio entre y alrededor de los artículos flexibles justify-content
