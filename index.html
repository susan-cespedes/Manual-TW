ELEMENTOS HTML

Elemento Descripcion Documentacion
<html> Representa la raiz del documento html
<div> Elemento generico div
<table> Representa una tabla table
<p> Distribuye el texto en párrafos p
<input> Crea controles interactivos para formularios basados en la web input
<head> Información general (metadatos) acerca del documento head
<base> Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas de un documento base
<link> Especifica la relación entre el documento actual y un recurso externo link
<meta> Eportar información sobre el documento meta
<noscript> Aporta contenido alternativo al elemento script noscript
<script> Se utiliza para incrustar código o datos ejecutables script
<style> Contiene información de estilo para un documento o parte de un documento style
<title> Define el título del documento que se muestra en la barra de título del navegador o en la pestaña de una página title
<a> Crea un hipervínculo a páginas web, archivos, direcciones de correo electrónico, ubicaciones de página o cualquier otra cosa que una URL pueda abordar a
<abbr> Representa una abreviatura o un acrónimo abbr
<address> Indica que el HTML proporciona información de contacto para una persona o para una organización address
<article> Representa una composición independiente en un documento, página, aplicación o sitio, que es de forma independiente article
<aside> Representa una parte de un documento cuyo contenido está indirectamente relacionado con el contenido principal del documento aside
<audio> Se utiliza para incrustar contenido de sonido en los documentos audio
<b> Se utiliza para llamar la atención del lector sobre el contenido del elemento b
<bdo> Anula la direccionalidad actual del texto bdo
<bdi> Dice al algoritmo bidireccional del navegador que trate el texto que contiene de forma aislada del texto que lo rodea bdi
<blockquote> Indica que el texto adjunto es una cita extendida blockquote
<br> Produce un salto de línea en el texto br
<button> Elemento interactivo activado por un usuario con un mouse, teclado, dedo, comando de voz u otra tecnología de asistencia button
<canvas> Con la API de secuencias de comandos de lienzo o la API de WebGL para dibujar gráficos y animaciones canvas
<cite> Se usa para describir una referencia a un trabajo cite
<code> Muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora code
<data> Vincula una determinada pieza de contenido con una traducción legible por máquina data
<detalist> Representan las opciones permitidas o recomendadas disponibles para elegir dentro de otros controles datalist
<del> Representa un rango de texto que se ha eliminado de un documento del
<details> Crea un widget de divulgación en el que la información es visible solo cuando el widget se cambia a un estado "abierto" details
<dfn> Indica el término que se define dentro del contexto de una frase u oración de definición dfn
<dl> Representa una lista de descripción dl
<em> Marca el texto que tiene énfasis em
<embed> Incrusta contenido externo en el punto especificado del documento embed
<fieldset> Agrupa varios controles, así como etiquetas dentro de un formulario web fieldset
<figure> Representa contenido autónomo, potencialmente con un título opcional figure
<footer> Representa un pie de página para su ancestro más cercano que secciona el contenido footer
<form> Representa una sección de documento que contiene controles interactivos para enviar información form
<h1> Nivel de sección más alto h1
<h2> Nivel de sección segundo h2
<h3> Nivel de sección tercero h3
<h4> Nivel de sección cuarto h4
<h5> Nivel de sección quinto h5
<h6> Nivel de sección mas bajo h6
<header> Representa contenido introductorio, normalmente un grupo de ayudas introductorias o de navegación header
<hgroup> Representa un encabezado y contenido relacionado hgroup
<hr> Representa una ruptura temática entre elementos a nivel de párrafo hr
<i> Representa un rango de texto que se separa del texto normal, como texto idiomático, términos técnicos, designaciones taxonómicas, etc i
<iframe> Representa un contexto de navegación anidado, incrustando otra página HTML iframe
<img> Incrusta una imagen en el documento img
<ins> Representa un rango de texto que se ha agregado a un documento ins
<kbd> Representa un tramo de texto en línea que indica la entrada textual del usuario desde un teclado, voz o cualquier otro dispositivo de entrada de texto kbd
<label> Representa un título para un elemento en una interfaz de usuario label
<main> Representa el contenido dominante del cuerpo de un documento main
<map> Se utiliza para definir un mapa de imagen map
<mark> Representa texto que está marcado o resaltado para fines de referencia mark
<math> Se utiliza para escribir una única fórmula matemática math
<menu> Se describe en la especificación HTML como una alternativa semántica menu
<meter> Representa un valor escalar dentro de un rango conocido o un valor fraccionario meter
<nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación nav
<noscript> Secuencia de comandos en la página no es compatible o si las secuencias de comandos están desactivadas en el navegador noscript
<object> Representa un recurso externo, que se puede tratar como una imagen, un contexto de navegación, etc object
<ol> Representa una lista ordenada de elementos ol
<output> Elemento contenedor en el que un sitio o aplicación puede inyectar los resultados output
<picture> Ofrece versiones alternativas de una imagen para diferentes escenarios de visualización picture
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML pre
<progress> Muestra un indicador que muestra el progreso de finalización de una tarea progress
<q> Indica que el texto adjunto es una cita breve en línea q
<ruby> Representa pequeñas anotaciones que se representan encima, debajo o al lado del texto base ruby
<s> Muestra el texto tachado o con una línea que lo atraviesa s
<samp> Se utiliza para encerrar texto en línea que representa la salida de muestra de un programa de computadora samp
<script> Se utiliza para incrustar código o datos ejecutables script
<section> Representa una sección independiente genérica de un documento section
<select> Representa un control que proporciona un menú de opciones select
<small> Representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independiente de su estilo small
<span> Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada span
<strong> Indica que su contenido tiene una gran importancia, seriedad o urgencia strong
<sub> Especifica el texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos sub
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos sup
<template> Mecanismo para contener HTML que no debe representarse inmediatamente cuando se carga una página template
<textarea> Representa un control de edición de texto sin formato de varias líneas textarea
<time> Representa un período específico en el tiempo time
<u> Representa un tramo de texto en línea que debe representarse de manera que indique que tiene una anotación no textual u
<ul> Representa una lista desordenada de elementos, normalmente representada como una lista con viñetas ul
<var> Representa el nombre de una variable en una expresión matemática o un contexto de programación var
<video> Incorpora un reproductor multimedia que admite la reproducción de video en el documento video
<wbr> representa una oportunidad de salto de palabra wbr
<area> Define un área dentro de un mapa de imagen que tiene áreas predefinidas en las que se puede hacer clic area
<body> Representa el contenido de un documento HTML body
<head> Contiene información legible por máquina sobre el documento, como su título, scripts y hojas de estilo head
