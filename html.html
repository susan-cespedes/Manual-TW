ELEMENTOS GENÉRICOS

Elemento Descripción Documentación
<div> Crea secciones o agrupa contenido div
<span> Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada span

ELEMENTOS SEMANTICOS

Elemento Descripción Documentación
<html> Representa la raiz del documento html
<head> Información general (metadatos) acerca del documento head
<body> Representa el contenido de un documento HTML body
<header> Representa el contenido introductorio o de navegación header
<article> Representa una composición independiente en un documento, página, aplicación o sitio, que es de forma independiente article
<section> Representa una sección independiente genérica de un documento section
<nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación nav
<aside> Representa una parte de un documento cuyo contenido está indirectamente relacionado con el contenido principal del documento aside
<footer> Representa un pie de página para su ancestro más cercano que secciona el contenido footer
<main> Representa el contenido dominante del cuerpo de un documento main

MANEJO DE TEXTO

Elemento Descripción Documentación
<h1>-<h6> Niveles de encabezados que va del h1 que es el más grande hasta el h6 que es el más pequeño heading
<p> Distribuye el texto en párrafos p
<br> Produce un salto de línea en el texto br
<hr> Representa una ruptura temática entre elementos a nivel de párrafo hr
<wbr> Representa una oportunidad de salto de palabra wbr
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos sup
<sub> Especifica el texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos sub
<strong> Indica que su contenido tiene una gran importancia, seriedad o urgencia strong
<em> Marca el texto que tiene énfasis em
<mark> Representa texto que está marcado o resaltado para fines de referencia mark
<q> Indica que el texto adjunto es una cita breve en línea q
<blockquote> Indica que el texto adjunto es una cita extendida blockquote
<cite> Se usa para describir una referencia a un trabajo cite
<dfn> Indica el término que se define dentro del contexto de una frase u oración de definición dfn
<abbr> Representa una abreviatura o un acrónimo abbr
<small> Representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independiente de su estilo small
<ins> Representa un rango de texto que se ha agregado a un documento ins
<del> Representa un rango de texto que se ha eliminado de un documento del
<s> Muestra el texto tachado o con una línea que lo atraviesa s
<code> Muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora code
<samp> Se utiliza para encerrar texto en línea que representa la salida de muestra de un programa de computadora samp
<kbd> Representa un texto en línea que indica la entrada textual del usuario desde un teclado, voz o cualquier otro dispositivo de entrada de texto kbd
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML pre
<address> Indica que el HTML proporciona información de contacto para una persona o para una organización address
<caracteres especiales> Podemos ingresar caracteres especiales como "<", ">", "&", etc. entity

TABLAS

Elemento Descripción Documentación
<caption> Especifica el título de una tabla caption
<table> Representa datos tabulares, datos en una tabla bidimensional compuesta por filas y columnas table
<thead> Define un conjunto de filas en el encabezado de las columnas de la tabla thead
<tbody> Encapsula un conjunto de filas de la tabla, indica que comprende el cuerpo de la tabla tbody
<tfoot> Define un conjunto de filas que resumen las columnas de la tabla tfoot
<tr> Define una fila de celdas en una tabla tr
<th> Define una celda como encabezado de un grupo de celdas de tabla th
<td> Define una celda de una tabla que contiene datos td

ENLACES E IMÁGENES

Elemento Descripción Documentación
<a> Crea un hipervínculo a páginas web, archivos,etc o cualquier otra cosa que una URL pueda abordar a
<img> Incrusta una imagen en el documento img
